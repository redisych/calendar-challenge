# Calendar challenge
Based on [TypeScript React Starter](https://github.com/Microsoft/TypeScript-React-Starter).  
Here is the app containing configs and dependencies needed for local build and running.

## How to install
```
$ npm i
```

## How to run
```
$npm run start
```

open in a browser
```
http://localhost:3000
```

## How to run unit tests
```
npm run test
```
